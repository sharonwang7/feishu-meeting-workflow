### 案例 1.1：梦想驱动的人才战略

**类型**：原话提炼  
**压缩比**：2.7:1

**原始转写稿**（320 字）：
```
"我跟大家回顾一下啊，有很多婷婷那个新晋升的总监，不是很知道我们想打造
一个什么样的团队。凡岛是一个梦想驱动的公司，我们不想要一般人，我们也
不想要对公司凝聚力不好的人，所以我们希望找到一群人里面，找到一群能为
公司走很远的人，跟我们紧紧抱在一起的人。我们希望找一群很有凝聚力的人，
要打硬仗，打很长久的仗。我们对他很要求，但是我们也爱他，我们也绝对不
会辜负让这群人吃亏。我们想找一群跟我们真正能走很远很远的人共同成长。
这个是凡岛早期用应届毕业生人才变革，构建我们这个组织在文化价值观上的
大背景和大基础。我们做人才变革，我们讨论了很久很久，我们想做这么一
个组织，所以才有了人事有很多很特殊的那个非常规的一些操作和动作。"
```

**最终版**（120 字）：
```markdown
**凡岛坚持梦想驱动的人才战略，寻找有凝聚力、能打硬仗的长期伙伴，通过高要求、高投入、高关怀打造与众不同的组织竞争力：** 凡岛是一个梦想驱动的公司，我们不想要一般人，我们也不想要对公司凝聚力不好的人，所以我们希望找到一群人里面，找到一群能为公司走很远的人，跟我们紧紧抱在一起的人。所以我们希望找一群很有凝聚力的人，要打硬仗，打很长久的仗。
```

**提炼技巧分析**：
| 操作 | 原始内容 | 处理后 | 说明 |
|:---|:---|:---|:---|
| 删除口语化 | "我跟大家回顾一下啊"、"婷婷那个" | 删除 | 删除所有口语化表达 |
| 删除冗余 | "我们想做这么一个组织，所以才有了..." | 删除 | 删除解释性冗余 |
| 提炼总结句 | 无 | 新增加粗总结句 | 前置总结句，让读者扫读即懂 |
| 保留核心 | "梦想驱动"、"不想要一般人"、"打硬仗" | 保留 | 保留核心观点和金句 |
| 嵌入正文 | 独立引用 | 嵌入正文作为论据 | 原话服务于结论，不是独立存在 |

### 案例 1.2：物流中心女孩案例

**类型**：案例容器 + 原话提炼  
**压缩比**：1.9:1

**原始转写稿**（280 字）：
```
"那第二种是说你刚才说到一点，就是说人的这个抗压有很多的因素，一方面
是工作的、生活的各方面的，是不是？那其实我当然记得是说我刚来，然后
我们产品供应部的所有的储备干部就约我开了个会，然后他们当时提的一个
很多问题，这些年轻干部就问到的是说平常遇到的压力很大，然后很多人，
对，像我们那里有个物流中心的女孩，就一遇到压力就哭，就几乎一周在那
里每天几乎都要哭一次，所以大家会一开始觉得，嗯，这个人任性特别差。
但是我在那个会上我跟大家说不是的，她愿意哭，她是只是以情绪表达出来
的。他哭完他能够找到，他哭完能擦干眼泪继续做，然后下次做得更好，这
就是任性的提高。"
```

**最终版**（150 字）：
```markdown
<quote-container>
案例：我们有个同事，一遇到压力就哭，每天几乎都要哭一次，所以大家一
开始会觉得，这个人韧性特别差。但是我会跟大家说不是的，哭是一种情绪
表达，她愿意哭，说明她找到了自己排泄情绪的端口，她哭完能擦干眼泪继
续做，然后下次做得更好，这就是韧性的提高。现在这个同事也没怎么哭了，
从周围人反馈给我就变得很好，比她前三个月来每天都要哭一次，现在已经
不哭了，然后我们也认为她的韧性也提高了很多。
</quote-container>
```

**提炼技巧分析**：
| 操作 | 原始内容 | 处理后 | 说明 |
|:---|:---|:---|:---|
| 删除冗余叙述 | "那第二种是说"、"你刚才说到一点"、"那其实我当然记得是说我刚来" | 删除 | 删除案例引入的冗余叙述 |
| 删除口语化 | "是不是"、"对"、"嗯" | 删除 | 删除口语化填充词 |
| 用 quote-container 标注 | 无 | 添加 quote-container | 明确标注为案例 |
| 保留核心洞察 | "哭是情绪表达"、"擦干眼泪继续做" | 保留 | 保留核心洞察和金句 |
| 补充结果 | 无 | 新增"现在这个同事也没怎么哭了..." | 补充案例结果，形成完整闭环 |

### 案例 1.3：跪着带团队的金句处理

**类型**：分类归纳 + 原话提炼  
**压缩比**：4:1

**原始转写稿**（约 800 字）：
```
"当然我告诉你，能跪也是要水平的，就是就能跪也是一种水平，就是能跪的
意思是你可以把你的管理上限再乘一个比例。很多人不愿意，他就没有这个
比例可以乘，如果你愿意，你就有这个比例可以乘，所以你管理的人的范围
就会让那前面有韧性的人依然为你所用。"

"委屈一下自己，当然，如果你想打造一个更有韧性的、更信任的团队，那这
种人可能第二种思路我坚决不留，就有一些干部，这个是一种思路，短期思
路，还有一种思路是我管的人必须要跟我的价值观高度契合，我让他去东就
去东，去西就去西，这是我的基本原则和准则，我认为这个是基本价值观。"

"那我要的人，跟我一起干活的人，我就希望大家同心同德，是吧？劲往一处
使，指哪打哪。这个也是一种你组建团队的方式，有部分干部是坚决不允许
干部在我面前嘚瑟，也不允许他的思想那么的游荡而自由。那我要的队伍就
是像军队一样，指哪打哪，也是一种思路。"

"有不少干部就是这样子，我不要多元化的，我，我不要你们那个，你只要听
我讲就行。那我也，我希望打一支非常有执行力的队伍，大家有没有听家雄
就属于偏这种属性的干部，可以这么理解吗？华玲也是属于偏这属性的干部，
不理你们，你要想东想西想是你的事，你去你该想的地方想去，不要离我远
一点点，是吧？别影响了我们这个团队，OK，也有非常有包容性的，海纳百
川，那就看你希望是什么路线了。"

"所以对的意思是我本来已经爱执行力行的，从我的内心出发，我不是很看得
上他。但是我愿意为了成事我调整我自己，我接纳他，我内心虽然真的很不
爽，但是他完全不知道。我包容了他，他就在我面前，不管怎么，那个是吧？
也活得很好，但我知道，按我的，按我惯性的风格，我一用力，他马上就走，
但我跪着让他留下来。"

"如果调整不了，这个叫做调整。我们刚才说调整别人的任性，调整不了就调
整一下自己。哈哈哈，我们带的同事呢？自己既然这么不成熟，那有什么办
法？那我们就让自己更成熟，以包容的他，等待他长大。你等不了他长大，
那你就要找多几个人挑选出来。"
```

**最终版**（约 200 字）：
```markdown
- **包容型**：如果你想打造一个更有韧性的、更信任的团队，管理者得委屈一下自己，学会"跪着"。**跪着带团队是一种管理姿态，能跪也是一种水平**，能跪代表着管理者可以把管理上限再乘一个比例。很多人不愿意，就管不了跟自己特性不符或者不足够成熟的人，那就没有这个比例可以乘，如果愿意，那么就会让有能力但不够成熟的人依然为你所用，我们就让自己更成熟，以包容同事等待他们长大。但也不能一直跪，因为留不住的人，就算一直跪，这个人也是留不住的。

- **执行型**：还有一些干部是第二种思路——坚决不留，要求管的人必须要跟自己的价值观高度契合，要求价值观高度契合，指哪打哪，不允许思想游荡，希望大家同心同德，这个也是一种你组建团队的方式，但需要有更多人可以挑，不然团队就不足够稳定和有韧性。
```

**提炼技巧分析**：
| 维度 | 原始 | 最终 | 技巧 |
|:---|:---|:---|:---|
| 结构 | 6 段混合论述 | 2 个清晰分类 | 分类归纳 |
| 字数 | ~800 字 | ~200 字 | 压缩 4:1 |
| 删除 | 具体举例（家雄、华玲） | - | 删除人名 |
| 删除 | "是吧"、"OK"、"哈哈哈" | - | 删除口语化 |
| 新增 | 无 | "**包容型**"、"**执行型**" | 小标题 |
| 补充 | 无 | "但也不能一直跪..." | 边界条件 |

## 案例库 3：原话处理 5 原则案例支撑（5 个）

### 案例 3.1：必要性原则

**原则**：只引用无法更好总结的内容

**示例**：
- 原始："凡岛是一个梦想驱动的公司，我们不想要一般人"
- 处理：保留（因为这句话无法用更好的方式总结）
- 理由：这是王爷的核心观点，原话引用最有力量

### 案例 3.2：嵌入原则

**原则**：原话嵌入正文，作为论据支撑

**示例**：
- ❌ 错误：独立引用块
- ✅ 正确：**总结句：** 原话作为支撑

### 案例 3.3：精简原则

**原则**：删除口语化表达

**示例**：
- 原始："我跟大家回顾一下啊"、"婷婷那个"、"是吧"
- 处理：全部删除

### 案例 3.4：上下文原则

**原则**：金句需要有解释和上下文

**示例**：
- 原始："跪着带团队是一种管理姿态"
- 处理：前加"如果你想打造..."，后加"能跪代表着..."

### 案例 3.5：比例原则

**原则**：原话≤20%，总结≥80%

**示例**：
- 原始 800 字 → 最终 200 字
- 原话约 40 字（20%），总结约 160 字（80%）
