## [3.0.0] - 2026-05-15

### ✨ V3.0 框架补全

#### 文档 E: 《历史关联与冲突速查》
- **三重检测逻辑**：
  - 重复检测：按标题关键词匹配历史会议
  - 矛盾检测：检查相反关键词（停止/继续、放弃/投入等）
  - 遗漏检测：检查上次 P1 任务是否被提及
- **会议匹配算法**：标题关键词相似度 + 参会人重合度
- **冷启动机制**：首次使用提示

#### 文档 F: 《历史待办执行状态速查表》
- 提取上次 P1 任务
- 计算逾期状态（已完成/进行中/已逾期/待开始）
- 逾期天数计算 + 影响评估
- 生成速查表（Markdown 表格格式）

#### 会前 24h 自动检测
- Heartbeat 触发检测
- 扫描 1-7 天前的历史会议
- 为每个会议生成文档 E/F
- 保存到 `memory/pre-meeting-docs/` 目录

#### 任务闭环设计
- 飞书任务创建指令生成
- 参数映射（summary/due/members 等）
- 人员姓名 → open_id 转换逻辑
- 逾期追踪逻辑
- 完整流程设计文档（task-closure.md）

## 版本说明

### 版本号规则
- **主版本号（Major）**：不兼容的 API 变更
- **次版本号（Minor）**：向后兼容的功能性新增
- **修订号（Patch）**：向后兼容的问题修正

### 版本阶段
- **V1.x**：初始探索阶段（关键词匹配）
- **V2.x**：功能完善阶段（六级分类 + 文档体系）
- **V3.x**：LLM 集成阶段（智能提取 + 任务闭环 + 会前关联）
